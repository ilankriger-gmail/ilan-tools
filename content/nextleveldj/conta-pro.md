<div>  conta pro                                </div>
<div><br></div>
<div><font face="STIXTwoMath-Regular">⏺</font> O Claude Code oferece duas formas de autenticação:</div>
<div><br></div>
<div>  1. Claude Pro/Max (via OAuth) - Usa sua assinatura existente</div>
<div>  2. API Key - Usa créditos da API separadamente</div>
<div><br></div>
<div>  Para conectar com sua conta Pro, você precisa fazer logout da sessão atual e fazer login novamente escolhendo a opção correta.</div>
<div><br></div>
<div>  Execute no terminal:</div>
<div><br></div>
<div>  claude logout</div>
<div><br></div>
<div>  Depois:</div>
<div><br></div>
<div>  claude login</div>
<div><br></div>
<div>  Durante o login, você verá opções como:</div>
<div>  - Claude App (claude.ai account) - Esta é a opção para usar sua conta Pro/Max</div>
<div>  - API Key - Para usar créditos de API</div>
<div><br></div>
<div>  Escolha a primeira opção (Claude App) para usar sua assinatura Pro. Isso vai abrir o navegador para autenticar com sua conta claude.ai.</div>
<div><br></div>
<div>  Nota importante: O uso do Claude Code com conta Pro/Max tem algumas limitações de uso mensal comparado ao acesso via API. Se você atingir o limite, ele vai te notificar.</div>
