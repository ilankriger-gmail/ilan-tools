# "Atue como um Especialista em Frontend e UX.

<div>&quotAtue como um Especialista em Frontend e UX.</div>
<div><b>Objetivo:</b> Criar uma página de pesquisa NPS (Net Promoter Score) com uma experiência imersiva idêntica ao Typeform (Single-Page Application vibe).</div>
<div><b>Stack:</b> [Insira sua stack aqui, ex: React + Tailwind + Framer Motion OU HTML/CSS/Vanilla JS puro].</div>
<div><b>Requisitos Funcionais Obrigatórios:</b><br></div>
<ol>
<li><b>One-Question-at-a-Time:</b> Apenas uma pergunta visível por vez, centralizada verticalmente e horizontalmente.</li>
<li><b>Navegação Fluida:</b> Ao responder, a próxima pergunta deve deslizar suavemente para a tela (slide-up ou fade-in).</li>
<li><b>Keyboard First:</b> O usuário deve conseguir navegar usando apenas o teclado (Setas para selecionar notas, Enter para confirmar, Tab para inputs de texto).</li>
<li><b>Auto-Advance:</b> Ao selecionar a nota (0-10), deve pular automaticamente para a próxima pergunta sem precisar clicar em 'OK'.</li>
</ol>
<div><b>Estrutura das Perguntas:</b><br></div>
<ol>
<li>NPS Score (0-10) - Botões grandes e clicáveis.</li>
<li>Feedback de Texto (Opcional) - &quotO que motivou sua nota?&quot.</li>
<li>Email (Opcional) - Validação em tempo real.</li>
<li>Tela de Agradecimento final.</li>
</ol>
<div><b>Estética:</b> Minimalista, fontes grandes, barra de progresso sutil no topo ou rodapé.</div>
<div>Crie o código completo focado na performance das animações.&quot</div>
