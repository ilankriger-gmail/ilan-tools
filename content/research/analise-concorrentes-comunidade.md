# Análise de concorrentes - Plataformas de comunidade

## Plataformas analisadas

### 1. Circle
- **Preço**: $89-399/mês
- **Pontos fortes**: UI clean, integrações, spaces
- **Pontos fracos**: Sem gamificação nativa, caro
- **Público**: Criadores de curso

### 2. Discord
- **Preço**: Grátis (Nitro opcional)
- **Pontos fortes**: Tempo real, voz, bots
- **Pontos fracos**: Curva de aprendizado, UI confusa para leigos
- **Público**: Gaming, tech

### 3. Skool
- **Preço**: $99/mês
- **Pontos fortes**: Gamificação nativa, simples
- **Pontos fracos**: Pouca customização, em inglês
- **Público**: Coaches, infoprodutores

### 4. Arena Te Amo (nós!)
- **Preço**: Grátis para membros
- **Pontos fortes**: Customizado, corações, identidade única
- **Pontos fracos**: Escala, features em desenvolvimento
- **Público**: Comunidade do Moço do Te Amo

## Conclusão

Nosso diferencial é a **identidade emocional**. Nenhuma plataforma tem a conexão que o Moço tem com sua audiência. O sistema de corações + indicação é nosso moat.

## Recomendação

Investir em **mobile** (app nativo) e **notificações push** para aumentar retenção.
