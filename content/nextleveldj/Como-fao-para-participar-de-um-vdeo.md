# Como faço para participar de um vídeo?

<div><b><h2>Como faço para participar de um vídeo?</h2></b></div>
<div><b><br></b></div>
<div><b><br></b></div>
<div>Todos os vídeos acontecem de forma espontâneana rua. Muitas vezes as pessoas participam simplesmente por estarem no lugar certo, na hora certa. Não existe inscrição prévia para esses vídeos. Se você me encontrar na rua, venha falar comigo, peça uma foto, mas nunca peça para participar, pode acontecer de eu correr atrás de você para gravar.</div>
<div>Através do formulário de envie o seu sonho, fazemos a seleção de alguns poucos casos por ano, desta forma, vamos te chamar via Whatsapp para ajudar a criar um vaquinha ou outro tipo de ação para realizar o seu sonho.</div>
