# Modelo de monetização para criadores

## O problema
Criadores brasileiros dependem demais de AdSense e publis. Precisam diversificar.

## Framework: Pirâmide de Receita

```
         /\
        /  \  Mentoria 1:1 (R$5k+)
       /----\
      /      \ Curso/Comunidade (R$197-997)
     /--------\
    /          \ Produtos digitais (R$47-97)
   /------------\
  /              \ Conteúdo grátis (audiência)
 /________________\
```

## Aplicação no Arena Te Amo

1. **Base**: Conteúdo gratuito do Moço (YouTube, Instagram)
2. **Entrada**: Comunidade gratuita com gamificação
3. **Conversão**: Área premium com conteúdo exclusivo
4. **Premium**: Mentorias e eventos

## Métricas esperadas
- 5% da audiência entra na comunidade
- 10% da comunidade converte para premium
- 2% compra mentoria
