# Desafios de Quiz e Trivia

> Organizado em 2026-01-28

## Perguntas Interessantes para Vídeos
- Qual é a pergunta que nunca pode ser respondida com "sim"? → **Você está dormindo?**
- Quanto tempo uma pessoa passa na vida esperando o semáforo? → **~6 meses**
- Como se chama o medo de palavras longas? → **Hipopotomonstrosesquipedaliofobia**
- Qual é o único osso do corpo não conectado a outro? → **Hioide (garganta)**
- Qual parte do corpo tem mais bactérias? → **A boca**
- Qual palavra em português tem todas as vogais? → **SEQUOIA**
- Quem nasce em Salvador é o quê? → **Soteropolitano**
- Qual é o músculo mais forte do corpo? → **Masseter (mandíbula)**
- Qual alimento causa mais alergia no Brasil? → **Leite de vaca**
- Qual é o objeto mais perdido no mundo? → **Caneta**
- Qual é a palavra mais usada no Brasil? → **"de"**
- Qual órgão consome mais energia? → **Cérebro (~20%)**
- Quantos músculos pra dar um passo? → **+200**
- Qual palavra tem todas as letras repetidas? → **ARARA**
