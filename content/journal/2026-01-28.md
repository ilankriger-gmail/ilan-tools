# ViralWave multi-plataforma

Finalmente! O ViralWave agora posta automaticamente em **Twitter, Facebook e Threads**. 

## Timeline

- **Manhã**: Integração Facebook Graph API
- **Tarde**: Integração Threads API  
- **Noite**: Testes end-to-end com campanha real

## Lições aprendidas

A API do Facebook é mais burocrática que a do Twitter, mas a documentação é melhor. Threads é a mais simples — basicamente a mesma estrutura do Instagram.
